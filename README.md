# Project Diva Controller

## Main features:
' - Cheap
' - Simple
' - Compact (fits 250x250 Printbeds)
' - 4 Main buttons
' - Fake Slidepad using buttons
' - Enter/Escape Keys


## What you need:
' - 1: Raspbery Pi Pico
' - 8: Choc switches
' - 10: WS2812B 5050SMD LEDs
' - 3D Printer
' - Soldering Iron
' - Wires
' - (OPTIONAL: Protoboard/Perfboard for easier wiring)
